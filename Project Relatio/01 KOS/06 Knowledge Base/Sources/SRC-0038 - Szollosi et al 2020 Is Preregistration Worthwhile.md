# SRC-0038

# Szollosi et al. 2020 — Is Preregistration Worthwhile?

## Draft Source Record

---

# 1. Identification
Opinion/Letter: Szollosi, A., Kellen, D., Navarro, D. J., Shiffrin, R., van Rooij, I., Van Zandt, T., & Donkin, C. (2020), "Is Preregistration Worthwhile?", *Trends in Cognitive Sciences* 24(2):94–95. A short critical argument that preregistration (a paradigmatic formal research constraint) is **not diagnostic of good science** and that rewarding a single method is the wrong target. It anchors the *failure-mode / over-reliance* side: formal constraints can be treated as a substitute for good theory and can be idolised.

# 2. Source Evaluation (KOS-0003 §6)
- **Authority:** High for a viewpoint — peer-reviewed venue (TiCS); authors are established cognitive scientists and methodologists. But it is a **short opinion piece**, not an empirical study.
- **Transparency:** High — the argument is explicit and its provocative framing is openly acknowledged by the authors.
- **Independence:** High — academic; represents a genuine minority-critique position within the open-science debate.
- **Intent:** Persuade / correct an over-claim. Explicitly argumentative.

# 3. Limitations (what this source does NOT establish)
- It is an **argument, not evidence**: it does not empirically demonstrate that preregistration *harms* research, only that its benefits are over-stated and mis-targeted.
- It does **not** deny that analytic flexibility is a problem (SRC-0037's finding stands); its target is the *inference* that a formal constraint fixes the deeper (theory-mapping) problem.
- It is contested; many methodologists disagree, and the piece drew published rebuttals. Using it, one must present it as **one side of a live disagreement**, not a settled verdict.

# 4. Key Content / Passages Used
- Core argument: the diagnosticity of statistical tests "depends entirely on how well statistical models map onto underlying theories"; improving statistical procedure "does little to improve theories when the mapping is weak." Preregistration constrains the procedure, not the theory.
- Prescription: "we should not promote and reward any one method, but rather good arguments and good theory" — i.e., do not let a formal constraint become the proxy for quality.

# 5. Relationships (STD-0004)
- `supports` CLM-0032 (over-reliance on a fixed formal method; constraint as poor proxy for the real goal).
- `supports` CLM-0033 (do not idolise any single method — the synthesis's caution).
