# SRC-0027

# Harkin et al. 2016 — Does Monitoring Goal Progress Promote Goal Attainment? (Meta-Analysis)

## Draft Source Record

---

# 1. Identification
Meta-analysis of experimental (randomised) tests of whether prompting people to **monitor progress toward a goal** causes greater goal attainment: Harkin et al. (2016), *Psychological Bulletin* 142(2):198–229. 138 studies, N ≈ 19,951. It is the strongest causal (randomised) evidence that self-monitoring is not merely correlated with success but promotes it.

# 2. Source Evaluation (KOS-0003 §6)
- **Authority:** High — meta-analysis in *Psychological Bulletin* (a top review outlet) by leading self-regulation researchers.
- **Transparency:** High — mediation analysis and moderators reported.
- **Independence:** High — academic.
- **Intent:** Inform.

# 3. Limitations (what this source does NOT establish)
- Effect on attainment is **moderate (d+ ≈ 0.40)**, not large.
- Goals span many domains (not only health/wellness); generalisation to long-term wellness maintenance is an inferential step.
- Does not establish that monitoring **sustains** behaviour after monitoring stops.

# 4. Key Content / Passages Used
- Interventions increased frequency of progress monitoring (**d+ ≈ 1.98**) and, in turn, **promoted goal attainment (d+ ≈ 0.40, 95% CI 0.32–0.48)**. *(Verified this session via the White Rose repository / APA release.)*
- The monitoring→attainment effect was **mediated** by increased monitoring frequency, and was **larger when progress was reported/made public and physically recorded** — directly relevant to accountability and self-tracking design.

# 5. Relationships (STD-0004)
- `supports` CLM-0023 (randomised evidence that monitoring promotes attainment; complements Michie 2009's meta-regression).
