# SRC-0015

# Goessl, Curtiss & Hofmann (2017) — HRV Biofeedback for Stress and Anxiety (Meta-Analysis)

## Draft Source Record

---

# 1. Identification
Meta-analysis of heart-rate-variability (HRV) biofeedback training — slow-paced breathing (~6 breaths/min) with real-time cardiac feedback — for stress and anxiety, in *Psychological Medicine* (2017).

# 2. Source Evaluation (KOS-0003 §6)
- **Authority:** High — peer-reviewed meta-analysis; senior author (Hofmann) is a leading clinical-psychology researcher.
- **Transparency:** High — standard meta-analytic reporting; within- and between-group effects distinguished.
- **Independence:** Moderate-High — academic, not device-vendor funded.
- **Intent:** Inform.

# 3. Limitations (what this source does NOT establish)
- **Small evidence base (24 studies, 484 participants)** — susceptible to small-study/publication-bias inflation.
- **No long-term durability data** — outcomes are at/near end of training; does not establish >6–12 month relief.
- Heterogeneous populations and control conditions; large pre-post effect partly reflects expectancy.

# 4. Key Content / Passages Used
- **24 studies, 484 participants.**
- **Pre-post within-group Hedges' g ≈ 0.81**; **between-group vs control g ≈ 0.83** on stress/anxiety. *(Verified at abstract/summary level this session.)*
- Mechanistic rationale: ~6 breaths/min maximises HRV amplitude (baroreflex resonance).

# 5. Relationships (STD-0004)
- `supports` CLM-0013 (large acute effect; **no durability data** is the load-bearing caveat).
