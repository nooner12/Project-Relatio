# SRC-0028

# Gollwitzer & Sheeran 2006 — Implementation Intentions and Goal Achievement (Meta-Analysis)

## Draft Source Record

---

# 1. Identification
Meta-analysis of **implementation intentions** — "if situation X arises, I will do Y" if-then plans that specify the when/where/how of goal striving: Gollwitzer & Sheeran (2006), *Advances in Experimental Social Psychology* 38:69–119. It is the primary evidence base for planning as a remedy to the intention-behaviour gap.

# 2. Source Evaluation (KOS-0003 §6)
- **Authority:** High — the foundational meta-analysis, by the originator of the construct and a leading meta-analyst; very heavily cited.
- **Transparency:** High — 94 independent tests, effect and process moderators reported.
- **Independence:** Moderate–High — the lead author is the construct's originator (a mild theoretical-investment consideration), offset by the large independent replication base.
- **Intent:** Inform.

# 3. Limitations (what this source does NOT establish)
- Many primary studies are **short-term / laboratory or single-target** behaviours; durable long-term wellness maintenance is less directly tested.
- Effect is **medium-to-large (d ≈ 0.65)** but heterogeneous; implementation intentions work best for goals people are already committed to and can be undermined by low commitment.
- It addresses **translating an existing intention into action**, not creating motivation where none exists.

# 4. Key Content / Passages Used
- Across **94 independent tests**, implementation intentions had a **positive effect of medium-to-large magnitude (d ≈ 0.65)** on goal attainment. *(Verified at authoring via KOPS/Konstanz, SciRP, NCI-hosted PDF; **re-verified 2026-07-11** (remediation pass) via KOPS/Konstanz, SciRP and Semantic Scholar — "94 independent tests … d = .65" confirmed verbatim.)*
